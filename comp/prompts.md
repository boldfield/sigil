# Cross-language comparison prompts

Each prompt is run against a fresh LLM session with a language-specific system prompt prefix (`comp/contexts/<lang>.md`). The prompt body itself is **language-neutral** — it never mentions Sigil, Python, or Go. The session produces a program; the per-language eval driver compiles + runs it and compares stdout to the oracle.

Each prompt carries:

- **Prompt:** the language-neutral problem statement.
- **Oracle (stdout):** required stdout bytes, whitespace-sensitive.
- **Oracle (exit):** required process exit code.
- **Notes:** what the prompt exercises, why it was chosen.

Naming: `C<NN>` for cross-language (parallels `P<NN>` in `spec/validation-prompts.md`).

---

## C01 — hello world

**Prompt:** Write a program that prints exactly the text `hello, world` followed by a newline, then exits with status 0.

**Oracle (stdout):**
```
hello, world
```

**Oracle (exit):** `0`

**Notes:** Smoke test. Calibrates the harness (any non-trivial failure rate here means the runner is broken, not the language).

## C02 — sum 1 to 100

**Prompt:** Write a program that computes the sum of the integers from 1 to 100 inclusive (1+2+3+...+100), prints the result on a single line, then exits with status 0.

**Oracle (stdout):**
```
5050
```

**Oracle (exit):** `0`

**Notes:** Basic iteration or recursion. Tests integer arithmetic + summation idiom.

## C03 — fibonacci(15)

**Prompt:** Write a program that defines a function computing the Nth Fibonacci number using the recurrence F(0)=0, F(1)=1, F(n)=F(n-1)+F(n-2). Call it with N=15, print the result on a single line, then exit with status 0.

**Oracle (stdout):**
```
610
```

**Oracle (exit):** `0`

**Notes:** Recursive definition. Fibonacci is a canonical first-recursion exercise.

## C04 — factorial(10)

## C05 — fizzbuzz 1 to 15

**Prompt:** Write a program that, for each integer N from 1 to 15 inclusive (in order), prints exactly one line containing:

- `FizzBuzz` if N is divisible by both 3 and 5
- `Fizz` if N is divisible by 3 but not 5
- `Buzz` if N is divisible by 5 but not 3
- the integer N as a string otherwise

Then exit with status 0.

**Oracle (stdout):**
```
1
2
Fizz
4
Buzz
Fizz
7
8
Fizz
Buzz
11
Fizz
13
14
FizzBuzz
```

**Oracle (exit):** `0`

**Notes:** Branching with multiple conditions in a deterministic order. Tests modulo, conjunction in conditionals, and integer-to-string conversion.

## C06 — primality test for 29

**Prompt:** Write a program that determines whether the integer 29 is prime (a positive integer greater than 1 with no divisors other than 1 and itself). Print exactly `prime` on a single line if it is prime; print exactly `not prime` on a single line otherwise. Then exit with status 0.

**Oracle (stdout):**
```
prime
```

**Oracle (exit):** `0`

**Notes:** Bounded loop with early-exit semantics, conditional output. Sigil has no `for` / `break` so this exercises recursive divisor-testing.

## C09 — max of a hardcoded list

**Prompt:** Write a program that finds the maximum value in the list of integers `[3, 1, 4, 1, 5, 9, 2, 6]`. Print the result on a single line, then exit with status 0.

**Oracle (stdout):**
```
9
```

**Oracle (exit):** `0`

**Notes:** Fold / reduce over a list. Sigil has `std/list::fold`; Python has `max()`; Go requires manual iteration. Useful for measuring how each language's stdlib affects LOC and idiom choice.

## C10 — Collatz steps for 27

**Prompt:** The Collatz sequence starting from N is generated by repeatedly applying: if the current value is even, divide it by 2; otherwise multiply by 3 and add 1. The sequence ends when the value reaches 1. Write a program that counts how many steps the sequence takes to reach 1 starting from N=27 (the answer is 111). Print the count on a single line, then exit with status 0.

**Oracle (stdout):**
```
111
```

**Oracle (exit):** `0`

**Notes:** Loop with state and branching, recursion in Sigil's case. Collatz(27) is a well-known landmark — long enough to surface integer-overflow bugs in 32-bit implementations (peaks at 9232), short enough to terminate quickly. Sigil's `Int` is 63-bit so overflow isn't a concern; in Python it's irrelevant; in Go `int` is platform-sized but at least 32-bit signed.
